# Fullstack ToDo App

## Description

Simple and efficient task management application designed to help you organize and keep track of your daily tasks.

## Key Features

- 📝 Add new tasks quickly
- ✅ Update task status: "Not Started", "Active", "Archived" 
- 📋 View and manage tasks in their current states

---

## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/SaidKamol0612/ToDo-Master.git
cd ToDo-App-Master
```

### 2. Set Up Environment Variables

Create a `.env` file in the `src` directory on `backend` and set the necessary variables (e.g. database URL, etc.).

---

## Backend Setup

### 1. Create and Active Virtual Environment

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 2. Install Dependencies

```bash
cd backend
pip install -r requirements.txt
```

### 3. Run the Backend

```bash
cd src
python run.py
```

---

## Frontend Setup

### 1. Install Dependencies

```bash
cd frontend
npm install
```

### 2. Run the Frontend

```bash
npm start
```

## Tech Stack

* Python + FastAPI for backend
* SQLite with `aiosqlite`
* React for frontend
* RESTful API
